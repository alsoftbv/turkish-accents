# Turkish Accents

A macOS input method that uses the US keyboard layout but has the capability to write Turkish accents quickly.

Tap `g` and you get `g`. Hold `g` past the system's key-repeat delay, it's replaced in place with `ğ`. Same for the rest of the Turkish set.

| Key | Hold |
| --- | ---- |
| u, U | ü, Ü |
| o, O | ö, Ö |
| c, C | ç, Ç |
| s, S | ş, Ş |
| g, G | ğ, Ğ |
| i, I | ı, İ |

## Installing

### From a release build

1. Download `turkish-accents.zip` from the latest GitHub release.
2. Unzip and copy `turkish-accents.app` into `~/Library/Input Methods/`.
3. **Log out and back in** (one-time only, macOS only scans that folder at login).
4. System Settings → Keyboard → Input Sources → **+** → English → **Turkish Accents** → Add.
5. Switch to it from the menu-bar input indicator or with your input-source shortcut.

### From source

Requires Xcode 15+ and macOS 12+.

1. `git clone` and open `turkish-accents.xcodeproj`.
2. **⌘B** to build.
3. The scheme's post-build action copies the signed bundle into `~/Library/Input Methods/` and restarts the input method.
4. First install still requires log out / log in. After that, subsequent rebuilds hot-swap via `killall` without needing to log out.

- **Icon must be square.** macOS requires input-source icons to be a square 16×16 image (with a 32×32 retina rep in the same TIFF). The wider rectangular pill that keyboard layouts (US, TR, …) show is generated by the system specifically for layouts and is not available to input methods. Apple's own IMEs (Japanese, Vietnamese, Korean, Tamil, …) all ship square icons for the same reason. If the icon looks shorter than a neighbouring TR/US pill in the menu, that's why.
- **Hold duration is system-wide.** The threshold is the user's *Delay Until Repeat* in System Settings → Keyboard. Can't be configured per-IME.
- **First install needs a logout.** `imklaunchagent` only scans the Input Methods folders at login; there is no working API to force a rescan at runtime (`TISRegisterInputSource` returns success but is a no-op for this purpose on modern macOS).
